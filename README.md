# 🎵 Spotify Clone Project

A sleek and responsive Spotify-like web player built with pure HTML, CSS, and JavaScript. This project allows users to play their favorite music tracks, browse through a playlist, and control playback with an intuitive interface.

## ✨ Features
- **Responsive Navigation**: Minimalist and clean navigation bar.
- **Dynamic Playlist**: Automatically populated songs with cover art.
- **Music Controls**: Play/Pause, Next Track, and Previous Track functionality.
- **Interactive Seekbar**: Real-time progress bar (the "line") that shows song progress and allows manual seeking.
- **Animated Play Indicator**: A GIF that appears when music is playing for a "living" UI feel.

## 🚀 How to Run Properly

Since this project uses modern JavaScript and local media files, it's best to run it through a local HTTP server to avoid browser security restrictions.

### Option 1: VS Code Live Server (Fastest for Windows)
1.  Open this folder in **VS Code**.
2.  Install the **Live Server** extension.
3.  Right-click `index.html` and select **"Go Live"**.

### Option 2: Using Terminal (Node.js required)
1.  Open the terminal in this directory.
2.  Run the following command:
    ```bash
    npx http-server
    ```
3.  Open the link provided (usually `http://127.0.0.1:8080`).

## 🛠️ Built With
- **HTML5**: For semantic structure.
- **CSS3**: For a modern and responsive user interface.
- **JavaScript (Vanilla)**: For player logic and interaction.
- **FontAwesome**: For beautiful music control icons.

## 📁 Project Structure
- `index.html`: The main page and structure.
- `style.css`: The visual design and animations.
- `script.js`: The heart of the player's music logic.
- `songs/`: Audio files (.mp3).
- `covers/`: Album art files (.jpg).
